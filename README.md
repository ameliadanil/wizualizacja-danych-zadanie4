# Zadanie 4 – Wizualizacja danych

## Temat
Wizualizacja rynku koncertów muzycznych w Polsce z wykorzystaniem biblioteki Plotly.

## Wykorzystane technologie
- Python
- Pandas
- NumPy
- Plotly

## Zawartość projektu
Projekt obejmuje:
- wczytanie i eksplorację danych,
- wykres słupkowy,
- wykresy liniowe,
- histogram,
- boxplot,
- scatter plot,
- mapę,
- subploty,
- wnioski z analizy.
